# Schüler Fragen

Eine Sammlung von interaktiven Fragen für Schüler, gehostet auf GitHub Pages.

## Fragen

- **Frage 1**: Entzündungszeichen (Latein)
- **Frage 2**: Wundarten zuordnen
- **Frage 3**: Dekubitus - Bedeutung und Ursache
- **Frage 4**: Dekubitus-Stadien zuordnen
- **Frage 5**: Druckstellen - gefährdete Körperstellen
- **Frage 6**: Moderne Wundversorgung (Alt vs. Neu)
- **Frage 7**: Dekubitusprophylaxe - Maßnahmen zuordnen

## GitHub Pages aktivieren

1. Gehe zu deinem Repository auf GitHub
2. Klicke auf **Settings** (Einstellungen)
3. Scrolle nach unten zu **Pages** (im linken Menü)
4. Unter **Source** wähle:
   - **Branch**: `main` (oder `master`)
   - **Folder**: `/ (root)`
5. Klicke auf **Save**
6. Warte ein paar Minuten, bis GitHub Pages die Seite erstellt hat
7. Deine Seite ist dann unter `https://[dein-username].github.io/schueler-fragen/` erreichbar

## Lokale Vorschau

Um die Seiten lokal zu testen, öffne einfach die HTML-Dateien in deinem Browser oder nutze einen lokalen Server:

```bash
# Mit Python 3
python -m http.server 8000

# Mit Node.js (http-server)
npx http-server
```

Dann öffne `http://localhost:8000` im Browser.
